## Getting Started

This is a pnpm workspace for organizing modular packages.

### Quick Start

```sh
# Install dependencies
pnpm install

# Run tests across all packages
pnpm test

# Build all packages
pnpm build

# Lint all packages
pnpm lint
```

### Prerequisites

- Node.js 20+
- pnpm
